# Hero Background Image

Replace this file with your own hero background image.

## Recommended specs:
- **Format:** JPG or PNG
- **Size:** 1920x1080px (Full HD) or larger
- **File size:** Under 500KB for fast loading
- **Aspect ratio:** 16:9

## Where to find free hero images:
- [Unsplash](https://unsplash.com/) - Free high-quality photos
- [Pexels](https://www.pexels.com/) - Free stock photos
- [Pixabay](https://pixabay.com/) - Free images and videos

## Tips:
- Choose an image that represents your work/personality
- Make sure it has enough contrast for the text overlay
- Landscape orientation works best
- Consider using a subtle gradient overlay if the image is busy
